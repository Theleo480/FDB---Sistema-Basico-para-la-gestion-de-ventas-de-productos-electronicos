# Sistema Básico para la Gestión de Ventas de Productos Electrónicos

## Descripción
Este proyecto consiste en el desarrollo de un sistema de base de datos para la gestión básica de información sobre los productos vendidos en una mini empresa.

El sistema permitirá llevar un control del inventario, las ventas y los productos comercializados, facilitando la organización y el manejo de la información.

## Motivación
La motivación de este proyecto surge de la necesidad real de contar con un sistema que permita gestionar las ventas y el inventario, ya que actualmente no se dispone de una herramienta que realice estas funciones de manera adecuada.
